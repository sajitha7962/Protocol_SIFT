# Protocol SIFT - Frontend

React application serving as the Investigation Workspace.

## Directory Structure
- `src/pages`: Top-level route components
- `src/components`: Reusable UI elements grouped by domain
- `src/layouts`: App shells and structural wrappers
- `src/stores`: Zustand state management
- `src/hooks`: Custom React hooks

## Development

```bash
npm install
npm run dev
```
